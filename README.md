# Test-Driven Development

> In this project, we use TDD to develop a class where we write the tests before the code.

## Built With

- Ruby

## Getting Started

### Prerequisites
#### Setup
Cloning a repository

- On GitHub.com, navigate to the main page of the repository;

- Above the list of files, click "Code" button;

- Copy the URL to clone the repository.

- To clone the repository using HTTPS : https://github.com/fibini/TDD.git

- To clone the repository using an SSH key, including a certificate issued by your organization's SSH certificate authority : git@github.com/fibini/TDD.git

- To clone a repository using GitHub CLI : gh repo clone fibini/TDD

- Open Terminal

- Change the current working directory to the location where you want the cloned directory

- Type git clone, and then paste the URL you copied earlier
